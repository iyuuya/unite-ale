==============================================================================
INTRODUCTION						*unite-ale-introduction*

*unite-ale* is a |unite.vim| plugin for |ale|(Asynchronous Lint Engine).

Requirement:
- |unite.vim| (Recommend latest version at https://github.com/Shougo/unite.vim)
- |ale| (Recommend latest version at https://github.com/w0rp/ale) 

Latest version:
https://github.com/iyuuya/unite-ale


==============================================================================
USAGE							*unite-ale-introduction*

>
	:Unite ale
<
